# TimerApp-Android_Java

# 開発環境
* Android Studio 4.0
* Java 14.0.1

# 主な機能
* 「スタート」を押すと30秒のタイマーが始まる。
* 「一時停止」を押すと、タイマーがストップする。
* 「リセット」を押すとタイマーがリセットされる。
* タイマー終了後は、「リセット」を押すと再度タイマーを実行できるようになる。
